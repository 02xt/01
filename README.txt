# موقع شركة TUZ للصيانة والمبيعات

## كيفية استخدام موقع TUZ

تم تصميم هذا الموقع ليكون بسيطًا ويسهل تثبيته على أي استضافة. الموقع مبني بالكامل باستخدام HTML/CSS/JavaScript بدون أي تعقيدات برمجية إضافية.

### محتويات الموقع:
- index.html: الملف الرئيسي للموقع
- tuz-logo.png: شعار الشركة
- hero-bg.jpg: صورة الخلفية للقسم الرئيسي
- gallery1.jpg, gallery2.jpg, gallery3.jpg, gallery4.jpg: صور معرض الأعمال
- robots.txt: ملف توجيهات لمحركات البحث
- sitemap.xml: خريطة الموقع لمحركات البحث
- .htaccess: ملف إعدادات للاستضافة (Apache)

## كيفية رفع الموقع على الاستضافة:

1. قم بتحميل جميع ملفات الموقع من Replit.
2. تسجيل الدخول إلى لوحة تحكم الاستضافة (cPanel أو أي لوحة تحكم أخرى).
3. انتقل إلى مدير الملفات أو قم باستخدام FTP.
4. انتقل إلى المجلد الرئيسي للاستضافة (عادة يكون public_html أو www).
5. قم برفع جميع الملفات إلى هذا المجلد.
6. تأكد من أن index.html موجود في المجلد الرئيسي.

## للتخصيص:

إذا كنت ترغب في تعديل الموقع:
- لتعديل المحتوى النصي: افتح ملف index.html بأي محرر نصوص وقم بتعديل النصوص مباشرة.
- لتغيير الألوان: ابحث عن قسم `:root` في بداية CSS واستبدل قيم الألوان.
- لاستبدال الصور: استبدل الملفات الموجودة بنفس الأسماء.
- لتعديل معلومات الاتصال: قم بتحديث أرقام الهاتف وروابط وسائل التواصل الاجتماعي في ملف index.html.

## نموذج الاتصال:

نموذج الاتصال في الموقع الحالي يعمل بشكل تجريبي فقط (يعرض رسالة نجاح دون إرسال بريد فعلي).
إذا كنت ترغب في تفعيل نموذج الاتصال:

1. قم بإنشاء ملف PHP (على سبيل المثال: contact.php) وأضف كود لمعالجة النموذج.
2. عدل السطر التالي في ملف index.html:
   ```html
   <form id="contactForm">
   ```
   إلى:
   ```html
   <form id="contactForm" action="contact.php" method="post">
   ```
3. قم بتعديل سكريبت JavaScript لإزالة كود المحاكاة.

## ملاحظات هامة:
- تأكد من تحديث رابط الخريطة ليشير إلى موقع المتجر الفعلي.
- قم بتحديث عنوان الموقع في ملف sitemap.xml.
- بمجرد رفع الموقع، قم بفحصه على مختلف الأجهزة والمتصفحات للتأكد من عمله بشكل صحيح.

للمزيد من المساعدة، يمكنك التواصل مع مصمم الموقع.
